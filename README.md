# BINNII ♻️  
### AI-Powered Waste Management iOS App 🚧 (In Process)

An AI-powered iOS application that classifies waste using image recognition to promote sustainable recycling behavior.

## 📱 Overview
BINNII is a mobile application developed as part of a senior thesis project.  
The goal of this app is to promote environmental awareness by helping users correctly separate waste using AI-based image recognition.

## 🚀 Features
- 📸 AI-powered waste classification using image recognition
- ♻️ Waste category suggestions (general / recyclable / organic)
- 🎯 Reward system to encourage sustainable behavior
- 🔐 Secure user authentication
- 🔄 Real-time data synchronization

## 🛠 Tech Stack
- **iOS:** Swift, SwiftUI  
- **Architecture:** MVVM  
- **Backend:** Supabase  
- **AI/ML:** Image Recognition  
- **Tools:** Git
